# 📸 Portfólio Storymaker — Maria Clara

Este repositório contém o código-fonte de uma **Landing Page Profissional** desenvolvida para a Storymaker e Criadora de Conteúdo, Maria Clara. O projeto foi construído com foco em estética, performance e conversão de novos clientes.

## 🔗 Link do Projeto
> **Acesse aqui:** https://repositoriostorymaker-7uknd529b.vercel.app/


## ✨ Sobre o Projeto
O site funciona como um portfólio digital moderno, destacando trabalhos em nichos como estética (unhas), casamentos e eventos esportivos (Gracie Barra). O design utiliza uma paleta de cores sofisticada (`#F5C97A` e `#D63A6A`) e tipografias que transmitem elegância e autoridade.

## 🛠️ Tecnologias e Recursos
- **HTML5 & CSS3:** Estruturação limpa e estilização avançada (Grid/Flexbox).
- **Design Responsivo:** Adaptado perfeitamente para dispositivos móveis (Mobile-First).
- **Google Fonts:** Uso estratégico das fontes *Playfair Display* e *Poppins*.
- **Hospedagem:** [Vercel](https://vercel.com) (Continuous Deployment).

## 🚀 Funcionalidades
- **Hero Section:** Mockup de smartphone simulando o consumo de conteúdo (Stories).
- **Galerias Dinâmicas:** Organização por nichos de atuação com efeitos de *hover*.
- **Citação de Branding:** Espaço dedicado à filosofia de trabalho da profissional.
- **CTA (Call to Action):** Rodapé otimizado com links diretos para contato (E-mail, Instagram, Telefone).
